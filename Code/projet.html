Mon projet

Pourquoi ces études

J'aimerais suivre le cycle de cybersécurité pour plusieurs raisons. La sécurité représente un enjeu majeur actuellement, car tous les systèmes doivent être protégés afin d'assurer la vie privée des utilisateurs et le bon fonctionnement des machines industrielles, entre autres. Travailler dans ce domaine serait pour moi un vecteur d'épanouissement, car j'ai déjà eu l'occasion de faire quelques essais en sécurité, notamment avec le site root.me, où j'ai pu réaliser mes premières injections SQL et participer à des CTF, ce qui est plutôt divertissant.

 Le cycle DevOps peut également être intéressant. J'apprécie l'idée de partir d'une feuille blanche pour résoudre une problématique. Effectuer la phase de réflexion avec l'algorithmie, rechercher une optimisation pour que le programme puisse fonctionner avec le moins de ressources possible, puis réaliser le programme, tout cela m'a déjà passionné. J'ai pu créer quelques programmes informatiques en utilisant des passerelles API ou pour développer un assistant vocal, et chaque expérience m'a beaucoup plu.

Je choisirai donc ces deux options, en plaçant la cybersécurité en premier choix. Cependant, dans les deux cas, je serai satisfait du parcours proposé, car ce sont deux domaines qui m'intéressent.
